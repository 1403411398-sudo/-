#  鸢尾花数据分类与可视化实验

## 实验介绍
本项目基于鸢尾花数据集，实现**分类器结果可视化**、**决策边界绘制**、**概率分布热力图展示**三大任务，通过不同模型与可视化方法，直观呈现分类算法的效果差异。


## 任务说明
### 任务1：多分类器结果可视化（`task1.py`）
- **功能**：选择`petal_length`和`petal_width`两个特征，训练4种分类器（带RBF/分箱/样条特征的逻辑回归、梯度提升树），输出**3个类别概率热力图 + 预测类别图**的4列布局。
- **输出效果**：
  - 每行对应一个分类器，列依次为Class 0/1/2的概率分布、最终预测类别
  - 样本点按真实类别（蓝=setosa、橙=versicolor、绿=virginica）标记
  - 热力图以“白→对应类别颜色”的渐变展示分类器置信度


### 任务2：3D决策边界可视化（`boundary.py`）
- **功能**：生成3特征的二分类数据集，训练逻辑回归模型后，在3D空间中绘制**数据点 + 线性决策平面**。
- **输出效果**：
  - 3D散点图（红/蓝区分两类样本）
  - 叠加半透明灰色决策平面，直观展示高维空间的线性分割效果


### 任务3：分类概率热力图展示（`probability.py`）
- **功能**：针对`sepal_width`与`petal_length`特征，绘制单个分类器的**3个类别概率分布 + 预测类别**的4列图。
- **输出效果**：
  - 每个类别对应“白→对应颜色”的概率渐变热力图
  - 叠加真实样本点，清晰呈现分类器对不同区域的概率判断
